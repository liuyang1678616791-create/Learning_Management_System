# Learning_Management_System
这是一个**个人学习管理系统**，项目简洁但有完整功能，能够展示全栈开发的各项技术栈。
### 核心功能
1. **首页** - 展示学习任务列表
2. **任务详情页** - 查看任务详情和完成状态
3. **简单的CRUD操作** - 创建、读取、更新、删除学习任务
4. **API展示** - 清晰展示前后端API调用流程

### 页面设计
- 页面1：学习任务列表（首页）
- 页面2：任务详情/编辑页面
- 额外：简单的统计面板）
